Extern CSS

We hebben tot nu toe gebruik gemaakt van CSS styling die in de pagina staat. Als je website uit meerdere pagina's bestaat dan wil je waarschijnlijk dezelfde stijl voor al je pagina's gebruiken. Dit kan door de CSS code in een extern bestand plaatsen en dat bestand in je pagina te includen.

Verplaats de styling voor je website naar een extern bestand.

  1. Maak een nieuw bestand aan in Notepad++.
  2. Verplaats de code die binnen de style tags van je HTML bestand staat naar het nieuwe bestand.
  3. Sla het bestand op in de map C:\Devoxx4kids\HTML\includes. Kies een naam zonder spaties die eindigt op .css
  4. Verwijder de style tags uit je HTML bestand.
  5. Voeg een include toe voor je CSS bestand zoals in voorbeeld 18.
