Website bouwen - deel 1

<HTML/>

HTML is een taal die je gebruikt om aan de browser te vertellen hoe hij de inhoud van je pagina aan de bezoeker van je website moet laten zien.

Met een element vertel je de browser wat hij met een stukje inhoud moet doen. Een element staat altijd tussen de tekens < en >. elementen komen voor in paren en zijn van toepassing op wat er tussen staat.

Zo kan je op de pagina bijvoorbeeld een titel tonen met het h1 element.

Maak nu je eigen webpagina.

  1. Open in de browser de volgende URL: https://jsbin.com.
  2. Bedenk een leuke titel voor jouw website zoals in voorbeeld 1.

Titels geven structuur aan je pagina, zo kan de bezoeker van je pagina meteen zien welke tekst op je pagina heel belangrijk is en welke tekst minder belangrijk.

Er zijn 6 niveaus voor titels. Probeer ze uit.

Links

We gebruiken het attribuut href om de browser te vertellen naar welke pagina de bezoeker moet gaan als hij op de link klinkt.

Maak een link in je tekst naar een andere website.

target="_blank" zorgt dat de link in een nieuwe pagina opent.

Andere kleuren!

Om kleuren te veranderen maken we gebruik van CSS. Dit is een stukje code waarmee je de stijl van je pagina kan aanpassen. Je kan voor elk type element een stijl toevoegen en dat plaats je binnen het style element.

Voeg onderstaand stukje code toe bovenaan de pagina:

Elke kleur heeft een eigen kleurencode (hexcode).

Je kan een kleurenkiezer gebruiken om de hexcodes voor verschillende kleuren te vinden.

Probeer meerdere kleuren uit en probeer ook de titels van kleur te veranderen.

Je kan ook de achtergrondkleur van de pagina veranderen.

Wil je sommige tekstblokken in een andere kleur? Dat kan door een class attribuut aan een element mee te geven. In de style element kan je dan voor die class een aparte stijl bepalen.

Markeer enkele paragrafen als belangrijk door de tekstkleur naar rood aan te passen.

Plaatjes

Om een afbeelding te tonen gebruik je de img element.

Het attribuut src gebruik om de browser te vertellen waar hij het plaatje kan vinden.

Om de grootte van een plaatje aan te passen kan je een class gebruiken.

Probeer zelf een leuk plaatje te vinden voor je site en pas het attribuut src aan.
