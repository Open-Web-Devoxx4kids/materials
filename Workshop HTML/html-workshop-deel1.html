Website bouwen - deel 1

<HTML/>

HTML is een taal die je gebruikt om aan de browser te vertellen hoe hij de inhoud van je pagina aan de bezoeker van je website moet laten zien.

Met een tag vertel je de browser wat hij met een stukje inhoud moet doen. Een tag staat altijd tussen de tekens < en >. Tags komen voor in paren en zijn van toepassing op wat er tussen staat.

Een pagina start altijd met de tag html. Zo weet de browser dat de pagina in de taal HTML geschreven is.

Binnen de head tags zet je informatie over de pagina zoals de paginatitel. De paginatitel wordt weergegeven tussen de title tags. Tussen de body tags komt de inhoud van de pagina. Dat is wat in de webbrowser zal worden getoond.

Maak nu je eigen webpagina.

  1. Maak een nieuw bestand aan in Notepad++.
  2. Kopieer en plak voorbeeld 2 in het bestand.
  3. Sla het bestand op in de map C:\Devoxx4kids\HTML. Kies een naam zonder spaties die eindigt op .html
  4. Open nu je pagina in Chrome of een andere webbrowser.

Waar zie je de titel van je pagina?

Tekst

Tekst hoort thuis in de body en plaats je in een paragraaf p tag.

Hebben jullie al bedacht waar je webpagina over zal gaan? Schrijf een stukje tekst!

Gebruik de F5 knop om de pagina in je browser te verversen zodat je je wijzigingen kan bekijken.

Titels

Er zijn 6 niveaus voor titels. Probeer ze uit.

Titels geven structuur aan je pagina, zo kan de bezoeker van je pagina meteen zien welke tekst op je pagina heel belangrijk is en welke tekst minder belangrijk.

Links

We gebruiken het attribuut href om de browser te vertellen naar welke pagina de bezoeker moet gaan als hij op de link klinkt.

Maak een link in je tekst naar een andere website.

Als je wilt dat de link in een nieuwe pagina opent dan kan je het attribuut target="_blank" toevoegen aan je link.

Lijsten

Er zijn twee soorten lijsten in HTML.

Probeer beide typen lijsten op je website. Wat is het verschil?

Andere kleuren!

Om kleuren te veranderen maken we gebruik van CSS. Dit is een stukje code waarmee je de stijl van je pagina kan aanpassen. Je kan voor elk type tag een stijl toevoegen en dat plaats je binnen de style tag in de head van je pagina.

Voeg onderstaand stukje code toe binnen de head tags van je pagina en kijk wat er gebeurd.

Elke kleur heeft een eigen kleurencode (hexcode).

Je kan een kleurenpikker gebruiken om de hexcodes voor verschillende kleuren te vinden.

Probeer meerdere kleuren uit en probeer ook de titels van kleur te veranderen.

Je kan ook de achtergrondkleur van de pagina veranderen.

Wil je sommige tekstblokken in een andere kleur? Dat kan door een class attribuut aan een tag mee te geven. In de style tag kan je dan voor die class een aparte stijl bepalen.

Markeer enkele paragrafen als belangrijk door de tekstkleur naar rood aan te passen.

Plaatjes

Om een afbeelding te tonen gebruik je de img tag.

Het attribuut src gebruik om de browser te vertellen waar hij het plaatje kan vinden.

Om de grootte van een plaatje aan te passen kan je een class gebruiken.

Kies een plaatje uit de map images en voeg deze toe aan je pagina. Je kan ook een plaatje van internet downloaden en in de map plaatsen.
